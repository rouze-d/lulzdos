LULZDOS : DDoS TOOL

Lulzddos is tool for takedown website or server. This DDoS tool combine from Hping3 and T50.

need:<br>
```
sudo gem install lolcat
sudo apt install hping3 t50 gnome-terminal
```


usage:<br>
```
sudo bash lulzdos.sh
```
run as root
